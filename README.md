# golang-scaffolding

A golang scaffolding for getting started new projects.

Contains the following example domains:
- [ ] Env Variables.
- [x] Config.
- [x] Auth.
- [x] JWT.
- [x] GraphQL.
- [ ] PostgreSQL.
- [ ] Type Safe SQL.
- [ ] Docker Compose.
- [ ] Live reload.
- [ ] Admin.
- [ ] Unit tests.
