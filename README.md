# golang-scaffolding

A golang scaffolding for getting started new projects.

#### Getting Started

```
docker-compose up
```

Contains the following example domains:
- [x] Env Variables.
- [x] Config.
- [x] Auth.
- [x] JWT.
- [x] GraphQL.
- [x] PostgreSQL.
- [x] Type Safe SQL.
- [x] Docker Compose.
- [x] Live reload.
- [x] Admin.
- [x] Unit tests.
